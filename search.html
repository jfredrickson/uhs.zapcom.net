Search the UHS Website

By typing in words and/or phrases to search for, you can find what you need on the UHS Website without having to go through every single page. You can also take a look at the UHS Site Index for a complete list of all available pages.

Text to Search For:
Boolean: Case

Examples

If you wanted to search for all pages containing either the word "student" or the word "teacher", you would select OR as a boolean, and type the words:

student teacher

If you wanted to search for pages containing the words "computer" and "internet" and "software", you would select AND as a boolean, and type the words:

computer internet software

This search CGI is from Matt's Script Archive, and has been slightly modified.
